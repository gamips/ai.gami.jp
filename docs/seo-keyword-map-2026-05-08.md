# ai.gami.jp ページ別キーワード設計 2026-05-08

## 方針

全ページで同じキーワードを繰り返すのではなく、ページごとに検索意図を分ける。

- トップページは、会社全体の入口として `AI導入支援`、`生成AI導入支援`、`AI開発会社` を担う。
- サービス一覧は、相談メニューの入口として `生成AI導入支援`、`AI開発サービス`、`RAG構築`、`AIエージェント導入支援` を整理する。
- カテゴリ詳細は、商談キーワードをそれぞれ受け持つ。
- 料金ページは、`AI導入支援 料金`、`AI開発 費用`、`月2万円` を担う。
- 会社情報と問い合わせは、会社名検索・相談直前検索を補強する。

## 今回の実装範囲

- 各ページのtitle、description、構造化データ説明文を上記の役割に合わせて調整。
- トップ、サービス一覧、料金、会社情報、問い合わせ、お知らせの表示文にも主要キーワードを自然に追加。
- 旧3階層目の独立ページは引き続きindex対象外とし、カテゴリ詳細の中でキーワードを受け持つ。
