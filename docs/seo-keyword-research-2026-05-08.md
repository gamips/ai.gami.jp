# ai.gami.jp SEOキーワード競合調査 2026-05-08

## 前提

対象サイト: `https://ai.gami.jp/`

Search Console APIで `2026-02-08` から `2026-05-07` を確認したところ、現時点の検索露出はほぼ指名検索のみ。

| クエリ | クリック | 表示回数 | 平均掲載順位 |
| --- | ---: | ---: | ---: |
| gami ai | 1 | 14 | 1.21 |
| gami | 0 | 18 | 13.17 |
| ai gami | 0 | 4 | 1.25 |
| gami.ai | 0 | 2 | 1.00 |

ページ別でもトップページが中心で、非指名キーワードの土台はまだ弱い。したがって、単に既存ページのmetaを少し変えるだけでは足りない。競合が取りに行っている「生成AI導入支援」「AIエージェント導入支援」「RAG構築」「AI業務自動化」などの商談直前キーワードに対応する専用ページ群が必要。

## 推奨ページ構成

既存の「AI × SaaS」「AI × Growth」「AI × Brand」はブランド表現としては良いが、検索キーワードとしては弱い。検索流入を取りに行くページでは、H1/titleに日本語の検索語をそのまま入れる。

### 1. `/services/ai-implementation/`

狙うキーワード: `AI導入支援`, `生成AI導入支援`, `中小企業 AI導入支援`

H1案: `中小企業向けAI導入支援`

内容:
- まずヒアリングで、AIで何をするか/しないかを整理
- 月2万円〜で小さく始められる
- ChatGPT/Claude/Gemini、業務フロー、社内ルール、プロンプト、運用改善まで
- 「ツールを入れて終わり」ではなく、使われる状態まで伴走

### 2. `/services/ai-agent/`

狙うキーワード: `AIエージェント導入支援`, `AIエージェント 業務自動化`

H1案: `AIエージェント導入支援`

内容:
- AIに任せる業務範囲の設計
- 権限、承認、人間レビューの設計
- n8n/Dify/ChatGPT/Claude/API連携などは必要に応じて記載
- 「1エージェント5万円」ではなく、業務単位で相談して設計する方針にする

### 3. `/services/rag-chatbot/`

狙うキーワード: `RAG構築`, `社内AIチャットボット`, `社内マニュアル AI検索`

H1案: `RAG構築・社内AIチャットボット開発`

内容:
- 社内マニュアル、FAQ、過去問い合わせ、議事録などを検索/要約できるAI
- セキュリティ、権限、ログ、更新運用の説明
- 小規模PoCから始める導線

### 4. `/services/ai-business-automation/`

狙うキーワード: `AI業務自動化`, `生成AI 業務改善`, `AI DX支援`

H1案: `AI業務自動化支援`

内容:
- 議事録、メール下書き、レポート、見積、問い合わせ一次回答
- 現場の既存フローを壊さずに導入
- 人間の確認ポイントを残す設計

### 5. `/services/ai-writing-seo/`

狙うキーワード: `AI SEO記事作成`, `AIライティング`, `AIマーケティング`, `AI検索対策`

H1案: `AIライティング・SEO記事作成支援`

内容:
- SEO記事、プレスリリース、SNS投稿、事例記事の下書き
- Search Consoleを見ながら改善
- AI検索/LLMO対策は過剰に言い切らず、引用されやすい会社情報・サービス情報・事例整理から始める

## 既存サイトで直すべき点

1. `AI × Growth / AI × Writing` の価格・schemaに「Monthly agent rental 5万円」の古い表現が残っているため、ユーザー方針どおり `AI導入支援 月2万円〜` に合わせる。
2. `Services` ページの見出しに `AI導入支援` が弱い。ブランド表現の上に、検索語としての説明を追加する。
3. `Price` ページに `AI導入支援 料金`、`月2万円〜`、`まずはヒアリング` を明示する。
4. FAQを増やす。例: 「何から相談できますか」「月2万円でどこまでできますか」「ChatGPTをすでに契約していますが相談できますか」「社内データは外に出ますか」「AIエージェントとRAGの違いは何ですか」。
5. 実績ページまたはセクションを強化する。`育つ見積り`、`SkipCart`、`SPAT` などを「AI導入/業務システム/業務改善」の文脈で整理する。

## 優先順位

### すぐやる

1. `/services/ai-implementation/` を新設する。
2. `/price/` と構造化データを `月2万円〜` 方針へ修正する。
3. `/services/` のH1/リード文に `AI導入支援`、`生成AI導入支援`、`AI開発` を自然に入れる。
4. FAQを追加して、ロングテールを拾う。

### 次にやる

1. `/services/ai-agent/` と `/services/rag-chatbot/` を追加する。
2. `AI導入支援とは`、`生成AI導入支援の料金相場`、`ChatGPTを導入したが使われない理由` の3記事を作る。
3. 事例ページを作る。

### 後回し

1. `生成AIコンサルティング` 単体のビッグキーワードで正面勝負。
2. 大企業向けのオンプレ/RAG/セキュリティ大規模構築ページ。
3. 業界別ページを大量に作ること。まずは中小企業/小規模導入に絞る。
